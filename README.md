# 📊 Exploratory Data Analysis Dashboard (Python)

This project is a **Python-based Exploratory Data Analysis (EDA) and Visualization Dashboard** developed using **Pandas, Matplotlib, and Tkinter**.
The application allows users to analyze datasets, generate visualizations, and explore insights through an interactive graphical interface.

The goal of this project is to make **data analysis simple and visual**, even for beginners who want to understand datasets quickly.

---

## 🚀 Features

* 📂 Load and analyze datasets
* 📊 Generate visualizations such as **Bar Charts**
* 🧮 Perform basic **Exploratory Data Analysis (EDA)**
* 🖥️ Simple **Tkinter GUI interface**
* 📈 Data visualization using **Matplotlib**
* 📓 Jupyter Notebook support for analysis

---

## ⚙️ Installation

1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/EDA-Dashboard.git
```

2️⃣ Navigate to the project folder

```bash
cd EDA-Dashboard
```

3️⃣ Install required libraries

```bash
pip install pandas matplotlib
```

---

## ▶️ Run the Project

Run the Tkinter GUI application:

```bash
python tk.py
```

Or run the notebook:

```bash
jupyter notebook tk.ipynb
```
